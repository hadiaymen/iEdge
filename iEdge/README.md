# iEdgeDetector 🚀

A modern, professional-grade MATLAB App Designer project designed to handle computer vision feature extraction cleanly and beautifully. Styled in a dynamic Mars Red & Space Charcoal palette.

## ✨ Features
- **Integrated UI:** All interactions and generated plots load asynchronously in a structured 2x2 modular matrix.
- **Robust Error Handling:** File-browser cancellation catching, chronological state validation, and UI alert dialogs.
- **Pre-Processing Layers:** Automatically handles RGB to Grayscale vector conversions and implements Gaussian Smoothing  prior to edge extraction for cleaner gradient maps.
- **Multi-Algorithm:** Visualizes the differences between **Sobel** (derivative approximations), **Canny** (multi-stage hysteresis thresholds), and **Prewitt** (discrete differentiation) side-by-side.

## 🛠️ Usage
1. Clone the repository or navigate to this folder in MATLAB.
2. Ensure you have the **Image Processing Toolbox** enabled in your MATLAB distribution.
3. Open MATLAB, open `iEdge`, and run the command: `iEdgeDetector`.
4. Upload an image, click **Apply Edge Detection**, and view the computer vision output instantly.
